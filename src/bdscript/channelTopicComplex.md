# $channelTopic[]
Returns the topic of a channel by it's ID.

## Syntax
```
$channelTopic[channelID]
```

### Parameters
- `channelID` `(Type : Snowflake || Flag : Required )` : the channel ID to return its topic.

## Example
```
$nomention
<#$noMentionMessage> topic is : $channelTopic[$noMentionMessage]
```
