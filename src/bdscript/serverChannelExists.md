# $serverChannelExists
Checks if the channel exists in the guild.

## Syntax
```
$serverChannelExists[channel ID]
```

### Parameters 
- `Channel ID` `(Type: Snowflake,String || Flag: Emptiable)`: Returns `true`, if it exists otherwise `false`.

## Example
```
$nomention
$serverChannelExists[566370477967147018]
```
