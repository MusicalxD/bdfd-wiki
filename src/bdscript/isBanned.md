# $isBanned

Returns 'true' if user is banned, otherwise it returns 'false'. Requires 'BAN_MEMBERS' permission

## Usage
```
$isBanned[user ID]
```

## Parameters 
- `user ID ` `(Type : Snowflake || Flag : Required)` :the ID to check if banned or no.

## Example
```
$nomention
$isBanned[$mentioned[1]]
```
